# Gregorius Drugworks

**Gregorius Drugworks** is a GregTech CEu addon introducing advanced psychoactive, neurochemical, and pharmacological systems designed for immersive progression and technical integration.

Built with a focus on modularity, balance, and Greg design philosophy, it expands the chemical and industrial landscape while remaining fully compatible with GregTech CEu environments.

---

## Overview

This addon extends the GregTech ecosystem with new chemically-oriented mechanics and systems built using native GregTech and KubeJS infrastructure. It is designed to integrate naturally into technical modpacks without disrupting progression flow.


---

## Requirements

* Minecraft 1.20.1
* Forge / NeoForge
* GregTech CEu
* KubeJS

---

## Installation

1. Download the latest release
2. Place the contents into your instance following the instructions
3. Launch the game

---

## Compatibility

Gregorius Drugworks is designed for compatibility with:

* GregTech CEu-based modpacks
* Custom progression packs
* Singleplayer and dedicated servers
